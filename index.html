Hi! I like low-level development, tinkering with systems and developing tooling to enhance developer experiences.

Some of my projects:

bake - an imperative build system

An imperative build system with caching. Simpler than make, more versatile than just. Allows writing a build script using a set of ordered instructions as opposed to a set of rules, enhancing the experience of using a build script as opposed to writing shell scripts.

x86-disasm - an x86 disassembler

Tiny, parallelizable and embeddable x86 disassembler supporting all general purpose, SSE, SSE2, SSE3, SSSE3, SSE4.1, SSE4.2, AVX and some AVX2 instructions. Includes a fully-functional CLI tool including features such as printing in either Intel or adapted AT&T syntax. Performs 10x as fast as objdump, or nasm which are the two most known disassemblers to date.
